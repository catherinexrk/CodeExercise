2. 存储方式
  * 链式存储

  ```
  struct TreeNode{
  	int val;
  	TreeNode * left;
  	TreeNode * right;
  	TreeNode(t) : val(t),left(nullptr),right(nullptr){};
  }
  ```

  

  * 线性存储

  > 父节点位置i  左孩子2*i+1 右孩子2*i+2

3. 遍历方式
   1. 深度优先遍历DFS：往深处走，遇到叶子节点往回走
      1. 前序遍历：中左右
      2. 中序遍历：左中右
      3. 后序遍历：左右中
   2. 广度优先遍历BFS：一层一层的去遍历
      1. 层序遍历



* 深度和高度的区别

深度：从根节点开始，从上往下数，根节点的深度为1/0**前序**

高度：从某一层的叶子节点开始，开始点高度为1/0，往上数依次递增 **后序** 

* 二叉搜索树
  * 左子树上所有的节点的值小于根节点的值
  * 右子树上所有的节点的值大于根节点的值
  * 左、右子树也分别是二叉搜索树
